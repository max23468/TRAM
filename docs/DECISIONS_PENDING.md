# Decisioni aperte TRAM

Questo registro contiene decisioni strutturali non ancora approvate.

Stato attuale: le decisioni sospese o future sono tracciate in `docs/DECISIONS.md`,
`docs/ROADMAP.md` e `docs/BACKLOG.md`; questo file diventa il registro operativo
quando una scelta richiede valutazione prima di essere promossa ad ADR.

Decisione aperta nota:

- Target e runbook deploy futuri: da definire prima di qualunque deploy non
  locale.

Quando una decisione viene approvata:

- creare o aggiornare un ADR in `docs/decisions/`;
- collegarlo da `docs/DECISIONS.md`;
- aggiornare `docs/TOOLCHAIN.md`, `docs/OPERATIONS.md` o roadmap se cambia il
  comportamento operativo.
